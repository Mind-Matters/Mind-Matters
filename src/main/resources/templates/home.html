Here at Mind-Matters we're creating an application for patients and providers that will allow them to document different events throughout their day. This data is then presented in a way that can be used to notice trends and patterns in the data.

Features

Our graph feature will gather the data submitted to the mood scale and show each user their mood over time. The data will be displayed in a line graph format because it will be easier for the provider to quickly assess their patient’s mood.

Users can create events on a calendar with drag-and-drop functionality. Users are able to see multiple events on the calendar view and edit them.

Users are able to track their moods on a sliding scale each day. Over time the data will be used by their provider to analyze their mood over time.
